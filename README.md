# Hand-controlled-cursor-HCC-
Control the mouse with your hands, litterally.
-
Coded by me, with ChatGPT's tips and debug. 
Note that this tool is using mediapipe, so, you need python 3.11 to run it:
1) install python 3.11 from a trusted source (https://www.python.org/downloads/release/python-3110/)
2) check the requirements for libraries to install
3) py -3.11 handtrackv6.py in the terminal to run
4) have fun and make sure your webcam is on!

!NOTE! this is only compatible with windows 7 to 11

# INSTRUCTIONS
- to move the cursor, just move your index finger
- to click, close your thumb and your index finger
- to quit, just click "q" with the camera window visible.
